# Smart Parking Management System

A full-stack Smart Parking Management System that allows users to:

- Discover nearby parking lots
- Check real-time slot availability
- Reserve parking slots
- Make online payments
- Access parking using QR codes
- View booking history

## Tech Stack

### Frontend

- React
- Vite
- Axios
- Socket.IO Client

### Backend

- Node.js
- Express.js
- Prisma ORM

### Database

- PostgreSQL

### External Services

- Razorpay
- Google Maps
